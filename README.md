# photoapp
A simple photo sharing app that utilizes Google's Firebase to authenticate users and store images.

### Technologies used:
<li>Firebase Storage</li>
<li>Firebase Firestore</li>
<li>Firebase Authentication</li>
<li>UIKit</li>
<li>UserDefaults</li>
<li>Interface Builder</li>
<li>Xcode</li>
<li>Swift</li>
